凭据安全说明

  • PAT 不会写入配置、磁盘或日志
  • 发现会话绑定当前管理员会话
  • 批量启用结束后立即从内存删除
  • GitHub 不支持账号密码认证

重要：GitHub 删除创建 Deploy Key 的 PAT 时，关联密钥可能同时失效。请保留该 PAT，或准备在此重新安装密钥。

高级：使用现有 SSH 私钥连接单个仓库

私钥不能枚举账户仓库，只能验证你明确填写的 SSH URL。仅支持无口令 OpenSSH 私钥。

上传账户级私钥可能授予服务器访问多个仓库的权限。
